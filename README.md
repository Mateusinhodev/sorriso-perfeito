# Sorriso Perfeito

Site profissional desenvolvido em **React**, **JavaScript** e **Tailwind CSS**, criado para apresentar serviços odontológicos com um design **clean, moderno e eficiente**.

## Design e Identidade Visual

* Sistema de design com paleta em **azul profissional** e **turquesa calmante**
* Layout responsivo e minimalista
* Imagens geradas especificamente para representar uma **clínica odontológica**

## Funcionalidades Implementadas

### Página Inicial

* Sessão **Hero** com imagem de impacto
* Lista de **benefícios** destacados
* **Cartões de serviços** utilizando componente reutilizável
* **CTA** estratégico para aumentar conversões

### Página de Contato

* Formulário validado com campos:

  * Nome
  * E-mail
  * Mensagem
* Exibição de informações de contato e suporte

### Componentes

* **Header** com navegação funcional
* **Footer** com dados da clínica
* **ServiceCard** reutilizável para serviços

## SEO e Performance

* Estrutura otimizada para SEO
* Navegação fluida e funcional

## Tecnologias Utilizadas

* **React**
* **JavaScript (ES6+)**
* **Tailwind CSS**
* **shadcn/ui** para componentes modernos
* **lucide-react** para ícones otimizados
* **Zod** para validação de formulários

Desenvolvido com cuidado para oferecer uma experiência clara, agradável e confiável ao usuário.
